About Dandelion Creative Coding


Dandelion Creative Coding is a browser-based IDE purpose-built for creative coding with p5.js. It offers an intuitive, visually oriented environment designed to make coding, debugging, and exploring interactive sketches easier and more enjoyable.


Though still in active development, Dandelion already includes powerful tools that go beyond basic code editors. Its step-through execution lets you pause and advance your sketches one frame at a time — ideal for debugging animations and interactions. The unique Editor View visualizes rendered elements even outside the canvas, giving you a broader perspective on your sketch. Future updates aim to further improve accessibility and creative freedom for all developers and artists.


Key Features

  • Syntax-highlighted code editor (powered by CodeMirror)
  • Run, pause, and step through your p5.js sketches frame-by-frame
  • Editor View to inspect all rendering elements, including off-canvas ones
  • Export your sketches as standalone web packages (via JSZip)
  • New: Mobile compatible UI

Technologies Used

  • p5.js – to run user-created sketches
  • CodeMirror – for live code editing with syntax highlighting
  • JSZip – to package and export projects as ZIP files
  • Prettier (via UNPKG) – for clean and consistent code formatting

Privacy & Licensing

Dandelion respects your privacy. It runs entirely in your browser with no backend services or data tracking. All code stays local and nothing is stored remotely.


⚠️ Security Notice: Dandelion uses basic heuristic rules to detect potentially unsafe JavaScript code. While it can block many common threats, it is not a secure sandbox and may miss advanced or obfuscated attacks. Do not use Dandelion to run untrusted code in production environments without additional sandboxing or isolation.
