About Dandelion Creative Coding


Dandelion Creative Coding is now a full-fledged creative coding platform built around p5.js. Beyond a simple editor, it provides an environment to explore, create, and interact with projects directly in your browser.


Though still in active development, Dandelion already includes powerful tools that go beyond basic code editors. Its step-through execution lets you pause and advance your sketches one frame at a time — ideal for debugging animations and interactions. The unique Editor View visualizes rendered elements even outside the canvas, giving you a broader perspective on your sketch. Future updates aim to further improve accessibility and creative freedom for all developers and artists.


The platform lets you import .js, .txt, and .json files, and create new .txt and .json files. You start at the main platform page with featured projects — many thanks to garabatospr for allowing us to showcase his p5 projects!



Key Features

  • New: Browser-based creative coding platform, not just an editor
  • New: Import and create .js, .txt, and .json files
  • Run, pause, and step through your p5.js sketches frame-by-frame
  • Preview to inspect all rendering elements, including off-canvas ones
  • Syntax-highlighted code editor (powered by CodeMirror)
  • Export your projects as standalone web packages (via JSZip) (Doesn’t work on mobile)
  • Mobile-compatible interface
  • New: Auto-save functionality that saves your project instantly after every change
  • New: Console with input support to evaluate code in your sketches
  • New: Gizmos functions: beginGizmos() and endGizmos() for rendering control
  • New: Projects now run inside a sandboxed iframe for safer execution
  • New: 404 and 503 error landing pages
  • Log: Several bugs fixed for smoother operation

Technologies Used

  • p5.js – to run user-created sketches
  • CodeMirror – for live code editing with syntax highlighting
  • JSZip – to package and export projects
  • Prettier (via UNPKG) – for clean code formatting
  • Acorn – for the previous code scanner

About the Developer

Dandelion is a solo project by Jonadab Moiseyev Olvera Marichal. While there’s no personal site yet, you can join the Dandelion Discord server or visit our Reddit page to connect, share feedback, and get involved in the community.


Privacy & Security

Dandelion runs entirely in your browser with no backend services; all code stays client-side.


⚠️ Warning: The safety scanner is currently inactive. While sketches run in a sandboxed iframe, evaluating untrusted code can still be dangerous. Avoid running production-level or untrusted code.
